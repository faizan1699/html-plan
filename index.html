HTML Tag Examples

<div> container

This is a div. It is a generic block container you can use to group other elements. 
<div>
  This is a generic container for grouping elements.
</div>

<nav>, <ul>, <li>

Below is another simple navigation menu made with a list:

  • Home
  • Lessons
  • Projects
<nav>
  <ul>
    <li>Home</li>
    <li>Lessons</li>
    <li>Projects</li>
  </ul>
</nav>

Lists: <ul>, <ol>, <li>

Unordered list (bullets):

  • Learn basic HTML tags
  • Practice Git commands
  • Upload code to GitHub
<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>

Ordered list (numbers):

  1. Install Git
  2. Create a repository
  3. Commit and push changes
<ol>
  <li>First step</li>
  <li>Second step</li>
</ol>

<table> example

A simple table comparing some HTML tags:

HTML Tags Comparison
Tag Type Short description
<div> Block Generic container for content.
<nav> Block Section that contains navigation links.
<ul>, <ol>, <li> Block Lists of items, unordered (ul) or ordered (ol).
<table> Block Displays data in rows and columns. 
<table border="1">
  <caption>Table Title</caption>
  <thead>
    <tr>
      <th scope="col">Header 1</th>
      <th scope="col">Header 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Data 1</td>
      <td>Data 2</td>
    </tr>
  </tbody>
</table>

Layout and Form Tags

Layout: <section>, <article>, <aside>, <hr>

Section title

This is an article inside a section. It could be a blog post or a single piece of content.
This is an aside. It can contain side notes or extra information. 
<section>
  <h2>Section Title</h2>
  <article>
    <p>Article content here</p>
  </article>
  <aside>
    <p>Side note or related info</p>
  </aside>
</section>

The horizontal line below is an <hr> tag, often used to separate content:

<span> and <br>

This is a span inside a paragraph, used for inline content.
This text is on the next line because of a <br> line break. 

<p>
  This is a <span>span element</span> inside a paragraph.
  <br />
  This is on a new line.
</p>

Form: <form>, <label>, <input>, <textarea>, <button>

Simple contact form (for practice only, it does not send data):




<form>
  <label for="name">Name:</label>
  <input id="name" type="text" name="name" />
  <br />
  <label for="message">Message:</label>
  <textarea id="message" name="message" rows="3" cols="30"></textarea>
  <br />
  <button type="submit">Send</button>
</form>

<select> and <option>

Choose a topic you want to learn:

<label for="topic">Choose topic:</label>
<select id="topic" name="topic">
  <option value="html">HTML</option>
  <option value="css">CSS</option>
</select>
