HTML Versions & Main Changes

Home | HTML History | HTML Versions

This page gives a quick overview of important HTML versions and what changed in each one, plus example code snippets inside <code> blocks.

HTML 1.0 (around 1991)

  • First public description of HTML by Tim Berners‚ÄëLee.
  • Very small set of tags: <h1>, <p>, <a>, <ul>, <li>, <img>.
  • No <table>, no forms, no CSS, no JavaScript.

Typical HTML 1.0 style page

<html>
  <head>
    <title>My first page</title>
  </head>
  <body>
    <h1>Welcome</h1>
    <p>This is simple text with a <a href="#">link</a>.</p>
  </body>
</html>

HTML 2.0 (1995)

  • Standardized the early HTML used on the web.
  • Added support for basic forms with <form>, <input>, <textarea>, <select>, <option>.
  • Became an official IETF standard (RFC 1866).

Example form in HTML 2.0 style

<form action="/submit" method="post">
  <p>Name: <input type="text" name="name"></p>
  <p>Message:<br>
    <textarea name="message"></textarea>
  </p>
  <input type="submit" value="Send">
</form>

HTML 3.2 (1997)

  • Added tables with <table>, <tr>, <td>, and more form elements.
  • Included many presentational tags like <font> and <center>, now replaced by CSS.
  • Reflected what browsers of that time (Netscape, IE) already supported.

Example table and presentational tags

<center>
  <font face="Arial" color="blue">My Table</font>
</center>

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Ali</td>
    <td>20</td>
  </tr>
</table>

HTML 4.01 (1999)

  • Improved structure and accessibility ideas.
  • Encouraged separating content (HTML) from presentation (CSS).
  • Defined three flavors: Strict, Transitional, and Frameset.

HTML 4.01 Strict DOCTYPE example

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN"
  "http://www.w3.org/TR/html4/strict.dtd">
<html>
  <head>
    <title>HTML 4.01 Example</title>
  </head>
  <body>
    <h1>Hello</h1>
  </body>
</html>

  • Re‚Äëwrote HTML 4.01 as XML ‚Äì very strict syntax rules.
  • Required lowercase tags, quoted attributes, and proper nesting, for example <br /> instead of <br>.
  • Popular for a while, but later HTML5 became more dominant.

XHTML 1.0 Strict example

  "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

    <p>All tags are closed properly.</p>
  </body>
</html>

HTML5 (from around 2010)

  • Current major version of HTML used on most modern websites.
  • Added many new features:
    • Semantic tags: <header>, <nav>, <section>, <article>, <footer>.
    • Built‚Äëin support for <audio> and <video>.
    • <canvas> and SVG support for graphics.
    • New form input types: email, url, date, range, color, etc.
  • Designed to work with CSS3 and JavaScript to build rich web apps.
