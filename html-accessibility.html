Accessibility in HTML

Home | Accessibility

What is Accessibility?

Accessibility (a11y) means making your website usable by everyone, including people with disabilities. HTML provides many features to help with this.

Semantic HTML

Use semantic tags instead of generic divs:

<!-- Good: Semantic -->
<header>
  <nav>...</nav>
</header>
<main>
  <article>...</article>
</main>
<footer>...</footer>

<!-- Bad: Generic divs -->
<div class="header">
  <div class="nav">...</div>
</div>

Alt Text for Images

Always provide descriptive alt text:

<!-- Good: Descriptive alt text -->
<img src="photo.jpg" alt="A red circle placeholder image" />

<!-- Decorative images should have empty alt -->
<img src="decoration.jpg" alt="" />

<!-- Bad: Missing alt attribute -->
<img src="photo.jpg" />

ARIA Attributes

aria-label

Provides a label when there's no visible text:

<button aria-label="Close dialog">X</button>

aria-labelledby

References another element that labels this one:

Contact Form

<h3 id="form-title">Contact Form</h3>
<form aria-labelledby="form-title">
  <input type="text" />
</form>

aria-describedby

References element that describes this one:

We'll never share your email

<label for="email">Email:</label>
<input id="email" type="email" aria-describedby="email-help" />
<p id="email-help">We'll never share your email</p>

aria-hidden

Hide decorative elements from screen readers:

<span aria-hidden="true">âœ¨</span>

aria-live

Announce dynamic content changes:

Status messages appear here
<div aria-live="polite" id="status">Status messages appear here</div>

Form Labels

Always associate labels with inputs:

<!-- Good: Label associated with input -->
<label for="username">Username:</label>
<input id="username" type="text" name="username" />

<!-- Also good: Wrapped label -->
<label>
  Username:
  <input type="text" name="username" />
</label>

Heading Hierarchy

Use headings in order (h1, h2, h3...):

<h1>Main Title</h1>
  <h2>Section Title</h2>
    <h3>Subsection</h3>
  <h2>Another Section</h2>

<!-- Bad: Skipping levels -->
<h1>Title</h1>
<h3>Subtitle</h3> <!-- Should be h2 -->

Keyboard Navigation

Make sure all interactive elements are keyboard accessible:

<!-- Good: Button is keyboard accessible -->
<button type="button">Click me</button>

<!-- Bad: Div with click handler (not keyboard accessible) -->
<div onclick="doSomething()">Click me</div>

<!-- Good: Use tabindex for custom elements -->
<div role="button" tabindex="0">Custom Button</div>

Skip Links

Allow users to skip to main content:

Skip to main content 
<a href="#main-content" style="position: absolute; left: -9999px;">Skip to main content</a>
<main id="main-content">
  <!-- Main content here -->
</main>

Language Attribute

Always specify the page language:

<html lang="en">
<!-- For Urdu -->
<html lang="ur" dir="rtl">

Table Accessibility

Use proper table structure:

Student Grades
Name Grade
John 95 
<table>
  <caption>Student Grades</caption>
  <thead>
    <tr>
      <th scope="col">Name</th>
      <th scope="col">Grade</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">John</th>
      <td>95</td>
    </tr>
  </tbody>
</table>

Complete Accessible Example

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <title>Accessible Page</title>
</head>
<body>
  <a href="#main">Skip to main content</a>
  
  <header>
    <nav aria-label="Main navigation">
      <ul>
        <li><a href="index.html">Home</a></li>
        <li><a href="about.html">About</a></li>
      </ul>
    </nav>
  </header>

  <main id="main">
    <h1>Page Title</h1>
    
    <article>
      <h2>Article Title</h2>
      <img src="photo.jpg" alt="Description of the image" />
      <p>Article content...</p>
    </article>

    <form aria-labelledby="form-title">
      <h2 id="form-title">Contact Form</h2>
      <label for="email">Email:</label>
      <input id="email" type="email" aria-describedby="email-help" required />
      <p id="email-help">We'll never share your email</p>
      <button type="submit">Submit</button>
    </form>
  </main>

  <footer>
    <p>© 2025 My Site</p>
  </footer>
</body>
</html>
