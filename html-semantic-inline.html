More Semantic Inline Elements

These tags give meaning to text, helping browsers and screen readers understand your content better.

<abbr> - Abbreviation

Use for abbreviations. Hover to see the full form: HTML 

<p>Learn <abbr title="HyperText Markup Language">HTML</abbr> today!</p>

<cite> - Citation

Use for titles of works (books, articles, etc.): The HTML Handbook is a great resource. 

<p><cite>The HTML Handbook</cite> is a great resource.</p>

<dfn> - Definition

Use for the term being defined: HTML stands for HyperText Markup Language. 

<p><dfn>HTML</dfn> stands for HyperText Markup Language.</p>

<kbd> - Keyboard Input

Use for keyboard keys or shortcuts: Press Ctrl + C to copy. 

<p>Press <kbd>Ctrl</kbd> + <kbd>C</kbd> to copy.</p>

<samp> - Sample Output

Use for sample output from a program: The program printed: Hello, World! 

<p>The program printed: <samp>Hello, World!</samp></p>

<var> - Variable

Use for variables in math or programming: The variable x equals 10. 

<p>The variable <var>x</var> equals 10.</p>

<time> - Date/Time

Use for dates and times: The event is on December 25, 2025. 

<p>The event is on <time datetime="2025-12-25">December 25, 2025</time>.</p>

With time:

Meeting at 2:30 PM on January 15. 

<p>Meeting at <time datetime="2025-01-15T14:30">2:30 PM on January 15</time>.</p>

<wbr> - Word Break Opportunity

Suggests where a long word can break if needed: supercalifragilisticexpialidocious 

<p>supercalifragilistic<wbr />expialidocious</p>

<mark> - Highlighted Text

Use to highlight important text: Remember to use semantic HTML for better accessibility. 

<p>Remember to use <mark>semantic HTML</mark> for better accessibility.</p>

<del> and <ins> - Deleted and Inserted Text

Use for showing edits: The price was $100 $80. 

<p>The price was <del>$100</del> <ins>$80</ins>.</p>

<sub> and <sup> - Subscript and Superscript

Subscript: H2O (water)

Superscript: E = mc2 (Einstein's equation) 

<p>H<sub>2</sub>O</p>
<p>E = mc<sup>2</sup></p>

Complete Example

In the book Web Development Guide, the term HTML is defined. To save a file, press Ctrl + S. The output will show: File saved successfully. The variable count tracks items. The deadline is June 30, 2025. Remember to use semantic tags for better SEO. 

<p>
  In the book <cite>Web Development Guide</cite>, the term
  <dfn>HTML</dfn> is defined. To save a file, press <kbd>Ctrl</kbd> +
  <kbd>S</kbd>. The output will show: <samp>File saved successfully</samp>.
  The variable <var>count</var> tracks items. The deadline is
  <time datetime="2025-06-30">June 30, 2025</time>. Remember to use
  <mark>semantic tags</mark> for better SEO.
</p>
