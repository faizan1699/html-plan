English – Short History of Web Development

1. Early Web (1990s)

  • Around 1990–1991, Tim Berners‑Lee created the World Wide Web at CERN, using HTML, HTTP, and the first web browser.
  • Early websites were very simple: mostly text, some images, and links.
  • Development was done with basic HTML files, no CSS or JavaScript.

2. Tables, Frames, and Early Scripts

  • In the mid‑1990s, developers used HTML <table> tags and frames to create page layouts.
  • JavaScript was introduced by Netscape to add interactivity in the browser.
  • CSS (Cascading Style Sheets) appeared to separate design from HTML structure.

3. Web Standards and CSS Layouts

  • Organizations like the W3C promoted web standards for HTML, CSS, and JavaScript.
  • Developers gradually moved from table‑based layouts to CSS‑based layouts using <div> and classes.
  • Browsers became more powerful and started to support more advanced features.

4. Web 2.0 and Dynamic Sites

  • Around the 2000s, “Web 2.0” described more interactive and social websites.
  • AJAX (Asynchronous JavaScript and XML) allowed pages to update without a full reload.
  • Server‑side languages like PHP, ASP.NET, Ruby, and later Node.js powered dynamic content.

5. HTML5, Responsive Design, and Modern Frameworks

  • HTML5 unified many features and added audio, video, canvas, and new semantic tags.
  • Responsive design with CSS media queries made sites work on phones, tablets, and desktops.
  • Frontend frameworks (like React, Angular, and Vue) made it easier to build large web apps.

6. Today

  • Modern web development combines HTML, CSS, JavaScript, and many tools (bundlers, package managers, frameworks).
  • But HTML is still the foundation — every page starts with structure in HTML.

اردو – ویب ڈویلپمنٹ کی مختصر تاریخ

۱۔ ابتدائی ویب (1990 کی دہائی)

  • تقریباً 1990–1991 میں ٹم برنرز لی نے سی ای آر این (CERN) میں ورلڈ وائڈ ویب بنایا، جس میں HTML، HTTP اور پہلا ویب براؤزر شامل تھا۔
  • ابتدائی ویب سائٹس بہت سادہ تھیں: زیادہ تر متن، چند تصاویر اور لنکس۔
  • اس وقت صرف HTML فائلز استعمال ہوتی تھیں، نہ CSS تھا نہ JavaScript۔

۲۔ ٹیبلز، فریمز اور شروع کا JavaScript

  • 1990 کی دہائی کے درمیان ڈویلپرز <table> اور فریمز کے ذریعے ویب پیج کا لیاؤٹ بناتے تھے۔
  • Netscape نے JavaScript متعارف کرایا تاکہ براؤزر کے اندر انٹرایکٹوٹی لائی جا سکے۔
  • CSS (Cascading Style Sheets) آیا تاکہ ڈیزائن کو HTML اسٹرکچر سے الگ رکھا جا سکے۔

۳۔ ویب اسٹینڈرڈز اور CSS لیاؤٹس

  • W3C جیسے اداروں نے HTML، CSS اور JavaScript کے لیے ویب اسٹینڈرڈز بنائے۔
  • آہستہ آہستہ ٹیبل بیسڈ لیاؤٹس چھوڑ کر CSS بیسڈ لیاؤٹس اپنائے گئے، جن میں <div> اور کلاسز استعمال ہوئیں۔
  • براؤزر زیادہ طاقتور ہوتے گئے اور نئی فیچرز کو سپورٹ کرنے لگے۔

۴۔ ویب 2.0 اور ڈائنامک سائٹس

  • 2000 کی دہائی کے آس پاس “Web 2.0” کی اصطلاح نئی انٹرایکٹو اور سوشل ویب سائٹس کے لیے استعمال ہونے لگی۔
  • AJAX کی مدد سے ویب پیج کا کچھ حصہ بغیر پورا پیج ری فریش کیے اپ ڈیٹ ہو سکتا تھا۔
  • سرور سائیڈ لینگویجز جیسے PHP، ASP.NET، Ruby اور بعد میں Node.js سے ڈائنامک ویب سائٹس بنیں۔

۵۔ HTML5، ریسپانسیو ڈیزائن اور جدید فریم ورکس

  • HTML5 نے کئی فیچرز کو ایک جگہ جمع کیا اور آڈیو، ویڈیو، کینوس اور نئے semantic ٹیگز شامل کیے۔
  • CSS media queries کے ذریعے ریسپانسیو ڈیزائن آیا، جس سے ویب سائٹس موبائل، ٹیبلٹ اور ڈیسک ٹاپ پر اچھا دکھائی دینے لگیں۔
  • فرنٹ اینڈ فریم ورکس (جیسے React، Angular، Vue) کی مدد سے بڑی ویب ایپلی کیشنز بنانا آسان ہو گیا۔

۶۔ آج کا دور

  • آج کی ویب ڈویلپمنٹ میں HTML، CSS، JavaScript کے ساتھ بہت سے ٹولز اور فریم ورکس استعمال ہوتے ہیں۔
  • لیکن بنیاد اب بھی HTML ہی ہے — ہر ویب پیج کی ساخت HTML سے ہی شروع ہوتی ہے۔
