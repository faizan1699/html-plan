HTML Links & Anchors

Home | Links

Basic Links

External link to another website:

Visit GitHub

<a href="https://github.com">Visit GitHub</a>

Link that opens in a new tab:

GitHub (new tab)

<a href="https://github.com" target="_blank">GitHub (new tab)</a>

Internal Page Links

Link to another page in your site:

Go to Home

<a href="index.html">Go to Home</a>

Link to a section on the same page (anchor):

Jump to Section 1

<a href="#section1">Jump to Section 1</a>
<!-- Later in the page -->
<h2 id="section1">Section 1</h2>

Section 1 (anchor target)

This section has id="section1" so links can jump here.

Email Links

Click to open email client:

Send Email

<a href="mailto:example@email.com">Send Email</a>

Email with subject and body:

Email with subject

<a href="mailto:example@email.com?subject=Hello&body=Hi there">Email with subject</a>

Phone Links

Click to call on mobile:

Call Us

<a href="tel:+1234567890">Call Us</a>

Download Links

Link that downloads a file:

Download README

<a href="README.md" download>Download README</a>

Download with custom filename:

Download as my-readme.txt

<a href="README.md" download="my-readme.txt">Download as my-readme.txt</a>

Link Relationships (rel attribute)

External link with rel="noopener noreferrer" for security:

Secure External Link

<a href="https://github.com" target="_blank" rel="noopener noreferrer">Secure External Link</a>

Link to next page:

Next Page

<a href="index.html" rel="next">Next Page</a>

Link to previous page:

Previous Page

<a href="index.html" rel="prev">Previous Page</a>

Link with Title Tooltip

Hover over this link to see a tooltip:

GitHub (hover me)

<a href="https://github.com" title="Visit GitHub homepage">GitHub (hover me)</a>

Disabled/Non-clickable Link

Link that doesn't go anywhere (using # or javascript:void(0)):

Placeholder Link

<a href="#">Placeholder Link</a>
<!-- or -->
<a href="javascript:void(0)">Non-clickable Link</a>
