Common layout tags

These tags help you organize the structure of a page:

Header area

Usually contains logo, title, and main navigation.

Navigation: list of links to sections or pages.

Section

A logical group of related content.

Article

Independent piece of content, like a blog post.

Aside

Side content such as notes, tips, or related links.

Footer

Appears at the bottom: copyright, contact, links.

Div and span

This is a div, a block container for grouping elements.

This sentence has a span element inside it, which is an inline container.
