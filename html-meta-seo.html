Meta Tags & SEO Basics

Home | Meta & SEO

What are Meta Tags?

Meta tags provide information about your webpage to browsers and search engines. They go inside the <head> section.

Basic Meta Tags

Character Encoding

Always include this first in your <head>:

<meta charset="UTF-8" />

Viewport (for mobile devices)

<meta name="viewport" content="width=device-width, initial-scale=1.0" />

Description

Short description of your page (shown in search results):

<meta name="description" content="Learn HTML with examples and tutorials" />

Keywords (less important now)

<meta name="keywords" content="HTML, web development, tutorial, learn" />

Author

<meta name="author" content="Your Name" />

Robots

Tell search engines how to index your page:

<meta name="robots" content="index, follow" />
<!-- Other options: noindex, nofollow -->

Open Graph Tags (for social media)

These tags help when your page is shared on Facebook, Twitter, etc.:

<meta property="og:title" content="My HTML Learning Site" />
<meta property="og:description" content="Learn HTML with examples" />
<meta property="og:image" content="https://example.com/image.jpg" />
<meta property="og:url" content="https://example.com" />
<meta property="og:type" content="website" />

Twitter Card Tags

For better previews when shared on Twitter:

<meta name="twitter:card" content="summary_large_image" />
<meta name="twitter:title" content="My HTML Learning Site" />
<meta name="twitter:description" content="Learn HTML with examples" />
<meta name="twitter:image" content="https://example.com/image.jpg" />

Refresh/Redirect

Auto-refresh page after 5 seconds:

<meta http-equiv="refresh" content="5" />

Redirect to another page after 3 seconds:

<meta http-equiv="refresh" content="3;url=https://example.com" />

Language

<html lang="en">
<!-- For Urdu: lang="ur" -->
<html lang="ur" dir="rtl">

Complete Head Example

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <meta name="description" content="Learn HTML with examples" />
  <meta name="author" content="Your Name" />
  <meta name="robots" content="index, follow" />
  
  <!-- Open Graph -->
  <meta property="og:title" content="My HTML Site" />
  <meta property="og:description" content="Learn HTML" />
  <meta property="og:type" content="website" />
  
  <title>My HTML Learning Site</title>
</head>
<body>
  <!-- Your content -->
</body>
</html>

SEO Best Practices

  • Use descriptive <title> tags (50-60 characters)
  • Write good meta descriptions (150-160 characters)
  • Use semantic HTML (header, nav, main, article, footer)
  • Add alt text to images
  • Use proper heading hierarchy (h1, h2, h3...)
  • Include keywords naturally in your content
<!-- Good title -->
<title>Learn HTML: Complete Guide with Examples</title>

<!-- Good description -->
<meta name="description" content="Master HTML with step-by-step tutorials, code examples, and interactive exercises. Perfect for beginners." />
