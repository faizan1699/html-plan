Embedding External Content

These tags let you embed content from other sources or embed other web pages.

<iframe> - Inline Frame

Embed another webpage inside your page:

<iframe
  src="https://example.com"
  width="600"
  height="400"
  title="Example iframe"
></iframe>

Iframe with Sandbox (for security)

<iframe
  src="https://example.com"
  sandbox="allow-scripts"
  width="600"
  height="400"
></iframe>

Embed YouTube Video

<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/VIDEO_ID"
  title="YouTube video player"
  frameborder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen
></iframe>

Embed Google Maps

<iframe
  src="https://www.google.com/maps/embed?pb=MAP_PARAMETERS"
  width="600"
  height="450"
  style="border:0;"
  allowfullscreen=""
  loading="lazy"
></iframe>

<object> - Embedded Object

Embed external resources like PDFs, Flash, or other plugins:

<object data="document.pdf" type="application/pdf" width="600" height="400">
  <p>Your browser doesn't support PDFs. <a href="document.pdf">Download it here</a>.</p>
</object>

Embed Flash (legacy, rarely used now)

<object data="animation.swf" type="application/x-shockwave-flash">
  <param name="movie" value="animation.swf" />
  <p>Flash not supported</p>
</object>

<embed> - Embed External Content

Similar to object, but simpler syntax:

<embed src="video.mp4" type="video/mp4" width="600" height="400" />

Embed PDF

<embed src="document.pdf" type="application/pdf" width="600" height="400" />

<param> - Parameters for Object

Used inside <object> to pass parameters:

<object data="video.swf" type="application/x-shockwave-flash">
  <param name="autoplay" value="true" />
  <param name="loop" value="false" />
</object>

When to Use Each

Tag Best For Example
<iframe> Embedding web pages, videos, maps YouTube, Google Maps, other websites
<object> PDFs, plugins, external resources PDF documents, Flash content
<embed> Simple embedding (less control) Videos, PDFs, images 
<!-- For web pages and videos: use iframe -->
<iframe src="https://example.com" width="800" height="600"></iframe>

<!-- For PDFs: use object or embed -->
<object data="file.pdf" type="application/pdf">
  <p>PDF not supported</p>
</object>

Security Considerations

  • Always use sandbox attribute with iframes when embedding untrusted content
  • Use title attribute for accessibility
  • Provide fallback content inside <object> for browsers that don't support it
  • Be careful with allowfullscreen - only use for trusted sources
<!-- Secure iframe example -->
<iframe
  src="https://trusted-site.com"
  sandbox="allow-scripts allow-same-origin"
  title="Trusted content"
  width="600"
  height="400"
>
  <p>Your browser doesn't support iframes.</p>
</iframe>
