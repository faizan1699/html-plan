SVG (Scalable Vector Graphics)

SVG lets you create vector graphics directly in HTML. They scale without losing quality.

Simple SVG Circle

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="black" stroke-width="2" fill="red" />
</svg>

SVG Rectangle

<svg width="200" height="100">
  <rect width="150" height="80" fill="blue" stroke="black" stroke-width="2" />
</svg>

SVG Line

<svg width="200" height="100">
  <line x1="10" y1="10" x2="190" y2="90" stroke="green" stroke-width="3" />
</svg>

SVG Polygon

<svg width="200" height="200">
  <polygon points="100,10 150,90 50,90" fill="yellow" stroke="black" stroke-width="2" />
</svg>

SVG Text

<svg width="200" height="50">
  <text x="10" y="30" font-family="Arial" font-size="20" fill="purple">Hello SVG!</text>
</svg>

SVG Path (Complex Shapes)

<svg width="200" height="200">
  <path d="M 10 80 Q 100 10, 190 80" stroke="orange" stroke-width="3" fill="none" />
</svg>

Canvas

Canvas is used for drawing graphics with JavaScript. It's a bitmap (pixel-based) drawing area.

Basic Canvas Element

<canvas id="myCanvas" width="200" height="100" style="border:1px solid #000000;"></canvas>

<script>
  var canvas = document.getElementById('myCanvas');
  var ctx = canvas.getContext('2d');
  ctx.fillStyle = 'red';
  ctx.fillRect(10, 10, 50, 50);
</script>

Note: Canvas requires JavaScript to draw. The example above shows the HTML structure. In a real page, you would add JavaScript to draw shapes, lines, and images.

Canvas Drawing Example (with inline script)

<canvas id="drawCanvas" width="200" height="100" style="border:1px solid #000000;"></canvas>
<script>
  var canvas = document.getElementById('drawCanvas');
  var ctx = canvas.getContext('2d');
  ctx.fillStyle = 'blue';
  ctx.fillRect(20, 20, 60, 40);
  ctx.strokeStyle = 'green';
  ctx.lineWidth = 3;
  ctx.strokeRect(100, 30, 50, 30);
</script>

SVG vs Canvas

Feature SVG Canvas
Type Vector (scalable) Bitmap (pixels)
JavaScript Optional Required
DOM Access Yes (each element) No (just the canvas)
Best For Icons, logos, simple graphics Games, animations, complex drawings 
<!-- SVG: Good for logos, icons, simple graphics -->
<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" fill="red" />
</svg>

<!-- Canvas: Good for games, animations, pixel art -->
<canvas id="gameCanvas" width="800" height="600"></canvas>
