Additional Advanced HTML Tags

These are some less commonly used but useful HTML tags for specific purposes.

<dialog> - Modal Dialog

Click the button to open a dialog:

This is a dialog box!

 
<dialog id="myDialog">
  <p>This is a dialog box!</p>
  <button onclick="document.getElementById('myDialog').close()">
    Close
  </button>
</dialog>

<!-- JavaScript to open -->
<button onclick="document.getElementById('myDialog').showModal()">
  Open Dialog
</button>

<menu> - Menu Element

    •  
    <menu>
  <li><button>Cut</button></li>
  <li><button>Copy</button></li>
  <li><button>Paste</button></li>
</menu>

    <template> - HTML Template

    Note: Template content is hidden. Use JavaScript to clone and display it. 

    <template id="myTemplate">
  <div>
    <h3>Template Content</h3>
    <p>This content is hidden until cloned.</p>
  </div>
</template>

<!-- JavaScript to use template -->
<script>
  var template = document.getElementById('myTemplate');
  var clone = template.content.cloneNode(true);
  document.body.appendChild(clone);
</script>

    <ruby>, <rt>, <rp> - Ruby Annotations

    Ruby annotations for pronunciation (common in Japanese, Chinese):

    漢 (kan) 字 (ji) 
    <ruby>
  漢 <rp>(</rp><rt>kan</rt><rp>)</rp>
  字 <rp>(</rp><rt>ji</rt><rp>)</rp>
</ruby>

    <bdi> - Bidirectional Isolate

    Example with mixed languages:

    User محمد wrote: "Hello, world!" 

    <p>
  User <bdi>محمد</bdi> wrote: "Hello, world!"
</p>

    <bdo> - Bidirectional Override

    Right-to-left text:

    This text reads from right to left

    Left-to-right text:

    This text reads from left to right

    <!-- Right-to-left -->
<bdo dir="rtl">This text reads from right to left</bdo>

<!-- Left-to-right -->
<bdo dir="ltr">This text reads from left to right</bdo>

    <slot> - Web Component Slot

    Note: <slot> is used with Web Components and requires JavaScript. Example shown in code only. 

    <!-- In a Web Component template -->
<template id="my-component">
  <div>
    <h2>Title</h2>
    <slot name="content">Default content</slot>
  </div>
</template>

<!-- Usage -->
<my-component>
  <span slot="content">Custom content here</span>
</my-component>
