HTML Tags & Use Cases

This page lists many important HTML tags with a short explanation of when you should use each one. It does not include every tag, but it covers the most common ones for basic web pages.

Document & root

  • <!DOCTYPE html> – tells the browser this is an HTML5 document.
  • <html> – root element that wraps the whole page.
  • <head> – metadata (title, charset, links, etc.).
  • <body> – all visible content on the page.

Head / metadata

  • <title> – text that appears in the browser tab.
  • <meta> – meta information, such as charset and viewport.
  • <link> – links external files like CSS (not used here).

Text structure

  • <h1>–<h6> – headings, from largest (h1) to smallest (h6).
  • <p> – paragraph of text.
  • <br> – line break inside text.
  • <hr> – horizontal rule, used to separate sections.

Inline text

  • <strong> – important, bold text.
  • <em> – emphasized (italic) text.
  • <span> – generic inline container for text.
  • <code> – inline code fragments.
  • <mark> – highlighted text.
  • <small> – smaller, side-note text.

Links & images

  • <a> – hyperlink to another page or site.
  • <img> – displays an image (needs src and alt).

Lists

  • <ul> – unordered (bullet) list.
  • <ol> – ordered (numbered) list.
  • <li> – list item inside ul or ol.

Layout & semantic structure

  • <header> – top area of a page or section (logo, title, nav).
  • <nav> – main navigation links.
  • <main> – main content of the document.
  • <section> – group of related content.
  • <article> – independent piece of content (like a blog post).
  • <aside> – side content, notes, or related links.
  • <footer> – bottom area, usually copyright and extra links.
  • <div> – generic block container for layout.
  • <span> – generic inline container (mentioned above too).

Tables

  • <table> – table wrapper.
  • <thead> – header row area.
  • <tbody> – body rows area.
  • <tr> – table row.
  • <th> – header cell.
  • <td> – normal data cell.
  • <caption> – title for the table.

Forms

  • <form> – wrapper for user input controls.
  • <label> – label for an input field.
  • <input> – single-line input (text, email, number, etc.).
  • <textarea> – multi-line text input.
  • <select> – drop-down list.
  • <option> – option inside a select.
  • <button> – clickable button (submit, reset, or normal).

Media

  • <audio> – audio player.
  • <video> – video player.
  • <source> – used inside audio/video to define media files.
