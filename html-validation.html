What is HTML Validation?

HTML validation checks your code against HTML standards to find errors and potential problems. Valid HTML works better across browsers and is more accessible.

Common HTML Mistakes

1. Missing Closing Tags

<!-- WRONG: Missing closing tag -->
<p>This paragraph is not closed
<p>Another paragraph

<!-- CORRECT: All tags properly closed -->
<p>This paragraph is closed</p>
<p>Another paragraph</p>

2. Incorrect Nesting

<!-- WRONG: Tags overlap incorrectly -->
<p>Text <strong>bold</p></strong>

<!-- CORRECT: Tags nest properly -->

3. Missing Alt Attributes on Images

<!-- WRONG: No alt attribute -->
<img src="photo.jpg" />

<!-- CORRECT: Alt attribute included -->
<img src="photo.jpg" alt="Description of the image" />

<!-- Also correct: Empty alt for decorative images -->
<img src="decoration.jpg" alt="" />

4. Missing Labels for Form Inputs

<!-- WRONG: Input without label -->
<input type="text" name="email" />

<!-- CORRECT: Input with associated label -->
<label for="email">Email:</label>
<input id="email" type="text" name="email" />

5. Using Deprecated Tags

<!-- WRONG: Deprecated tags (don't use these) -->
<font color="red">Text</font>
<center>Centered text</center>
<b>Bold</b> <i>Italic</i>

<!-- CORRECT: Use semantic HTML5 and CSS -->
<strong>Important text</strong>
<em>Emphasized text</em>
<!-- Use CSS for colors and centering -->

6. Missing DOCTYPE

<!-- WRONG: No DOCTYPE -->
<html>
  <head>...

<!-- CORRECT: HTML5 DOCTYPE -->
<!DOCTYPE html>
<html lang="en">
  <head>...

7. Incorrect Heading Hierarchy

<!-- WRONG: Skipping heading levels -->
<h1>Title</h1>
<h3>Subtitle</h3> <!-- Should be h2 -->

<!-- CORRECT: Proper hierarchy -->
<h1>Title</h1>
<h2>Subtitle</h2>
<h3>Sub-subtitle</h3>

HTML Best Practices

1. Use Semantic HTML5 Elements

<!-- GOOD: Semantic elements -->
<header>
  <nav>...</nav>
</header>
<main>
  <article>...</article>
</main>
<footer>...</footer>

<!-- AVOID: Generic divs everywhere -->
<div class="header">
  <div class="nav">...</div>
</div>

2. Always Include Alt Text for Images

<!-- Descriptive alt for informative images -->
<img src="chart.jpg" alt="Sales increased 25% in Q4" />

<!-- Empty alt for decorative images -->
<img src="decoration.jpg" alt="" />

3. Use Proper Form Labels

<!-- Method 1: Label with 'for' attribute -->
<label for="username">Username:</label>
<input id="username" type="text" />

<!-- Method 2: Label wrapping input -->
<label>
  Username:
  <input type="text" />
</label>

4. Indent Your Code Consistently

<!-- GOOD: Consistent indentation (2 spaces shown) -->
<div>
  <p>
    Text content here.
  </p>
</div>

5. Use Lowercase for Tags and Attributes

<!-- GOOD: Lowercase -->
<div class="container">...</div>

<!-- AVOID: Mixed case -->
<DIV CLASS="container">...</DIV>

6. Quote All Attribute Values

<!-- GOOD: Quoted attributes -->
<div class="container" id="main">...</div>

<!-- AVOID: Unquoted attributes -->
<div class=container id=main>...</div>

7. Close All Tags Properly

<!-- GOOD: All tags closed -->
<p>Paragraph</p>
<div>Container</div>

<!-- Self-closing tags in HTML5 -->
<img src="photo.jpg" alt="Photo" />
<br />
<hr />
<input type="text" />

8. Use Meaningful IDs and Classes

<!-- GOOD: Meaningful names -->
<div id="main-navigation">...</div>
<section class="article-content">...</section>

<!-- AVOID: Unclear names -->
<div id="div1">...</div>
<section class="box">...</section>

HTML Validators

Use these tools to validate your HTML:

  • W3C Markup Validator - Official HTML validator
  • HTML5 Validator - Modern HTML5 validator
  • Nu Html Checker - Another W3C validator

Accessibility Checklist

  • ✓ Use semantic HTML5 elements
  • ✓ Include alt text for all images
  • ✓ Use proper heading hierarchy (h1, h2, h3...)
  • ✓ Label all form inputs
  • ✓ Use lang attribute on html tag
  • ✓ Provide text alternatives for media
  • ✓ Ensure keyboard navigation works
  • ✓ Use ARIA attributes when needed

Complete Valid HTML5 Example

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <meta name="description" content="Page description" />
  <title>Page Title</title>
</head>
<body>
  <header>
    <h1>Site Title</h1>
    <nav aria-label="Main navigation">
      <ul>
        <li><a href="index.html">Home</a></li>
        <li><a href="about.html">About</a></li>
      </ul>
    </nav>
  </header>

  <main>
    <article>
      <h2>Article Title</h2>
      <p>Article content here.</p>
      <img src="photo.jpg" alt="Description of photo" />
    </article>
  </main>

  <footer>
    <p>© 2025 My Site</p>
  </footer>
</body>
</html>
