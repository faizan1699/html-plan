Tags That Go in <head>

The <head> section contains metadata and links to external resources. It's not visible on the page.

<title> - Page Title

Shows in the browser tab and search results:

<title>My HTML Learning Site</title>

<meta> - Metadata

We covered meta tags in the Meta & SEO page. Here are the basics:

<meta charset="UTF-8" />
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
<meta name="description" content="Learn HTML" />

<link> - External Resources

Link to CSS Stylesheet

<link rel="stylesheet" href="styles.css" />

Link to Favicon (icon in browser tab)

<link rel="icon" href="favicon.ico" type="image/x-icon" />
<!-- Or PNG -->
<link rel="icon" href="favicon.png" type="image/png" />

Preconnect to External Domain (for faster loading)

<link rel="preconnect" href="https://fonts.googleapis.com" />

DNS Prefetch

<link rel="dns-prefetch" href="https://cdn.example.com" />

Canonical URL (for SEO)

<link rel="canonical" href="https://example.com/page" />

Alternate Language Versions

<link rel="alternate" hreflang="ur" href="https://example.com/ur" />
<link rel="alternate" hreflang="en" href="https://example.com/en" />

<base> - Base URL

Sets a base URL for all relative links on the page:

<base href="https://example.com/" />
<!-- Now all relative links will use this base -->
<a href="about.html">About</a> <!-- Becomes https://example.com/about.html -->

Note: Use carefully - it affects ALL relative URLs!

<style> - Inline CSS

Add CSS directly in the HTML (not recommended for large styles):

<style>
  body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
  }
  h1 {
    color: blue;
  }
</style>

<script> - JavaScript

Inline JavaScript

<script>
  console.log('Hello from HTML!');
</script>

External JavaScript File

<script src="script.js"></script>

Script with Defer (loads after HTML)

<script src="script.js" defer></script>

Script with Async (loads in parallel)

<script src="script.js" async></script>

<noscript> - Fallback Content

Shows content if JavaScript is disabled:

<noscript>
  <p>Please enable JavaScript to use this website.</p>
</noscript>

Complete Head Example

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <meta name="description" content="Learn HTML with examples" />
  <meta name="author" content="Your Name" />
  
  <title>My HTML Learning Site</title>
  
  <link rel="icon" href="favicon.ico" type="image/x-icon" />
  <link rel="stylesheet" href="styles.css" />
  <link rel="canonical" href="https://example.com" />
  
  <style>
    /* Small inline styles if needed */
  </style>
  
  <script src="main.js" defer></script>
</head>
<body>
  <!-- Your content -->
</body>
</html>
