What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create pages on the World Wide Web.

Early days (1990s)

  • Around 1990, Tim Berners‑Lee created the first version of HTML while working at CERN.
  • In the early 1990s, HTML was very small and simple: basic text, links, lists, and images.
  • There was no CSS or JavaScript yet — pages were plain and mostly text.

HTML 2.0, 3.2, 4.01

  • HTML 2.0 (mid‑1990s) collected and standardized the early features of HTML.
  • HTML 3.2 and 4.01 added more tags, tables, forms, and better structure.
  • During this time, many presentation features (like font size and color) were done directly in HTML.

XHTML and strict structure

  • In the late 1990s and early 2000s, XHTML was created as a stricter version of HTML based on XML.
  • It required very clean code (proper closing tags, lowercase tags, and so on).

HTML5 and modern web

  • HTML5 started to become popular around 2010 and is the main version used today.
  • It added new semantic tags (<header>, <nav>, <section>, <article>, <footer>) and new features for audio, video, graphics, and forms.
  • HTML5 works together with CSS for styling and JavaScript for interactivity.

Why this matters for you

  • Modern HTML is more powerful and easier to read.
  • Semantic tags help browsers, search engines, and screen readers understand your pages better.
  • Learning HTML is still the first step to building modern websites and web apps.
