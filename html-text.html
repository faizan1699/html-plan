Paragraph and Inline Text

This is a normal paragraph of text. You can use <strong> to make text bold and <em> to make text italic. 

<p>
  This is a paragraph with <strong>bold</strong> and
  <em>italic</em> text.
</p>

Other inline tags: mark (highlight), code (inline code), u (underline), small (smaller text). 

<p>
  <mark>Highlighted text</mark>,
  <code>inline code</code>,
  <u>underlined text</u>,
  <small>smaller text</small>.
</p>

Common Mistakes Fixed

  • Mistake: Using <b> and <i> instead of <strong> and <em>
  • Fix: Use semantic tags. <strong> and <em> have meaning, while <b> and <i> are just visual.
  • Mistake: Skipping heading levels (h1 to h3)
  • Fix: Use headings in order (h1, h2, h3, h4...)
  • Mistake: Using <br> for spacing instead of paragraphs
  • Fix: Use <p> for paragraphs. Use <br> only for line breaks within a paragraph.
